Comandi Base Git

  • Come caricare i file da locale su web?
    1. Aprire la cartella contenenti i file
    1. Aprire il terminale WSL
    1. DIGITARE: git init
    1. DIGITARE: git remote add origin (link del repository)
    1. DIGITARE: git add .
    1. DIGITARE: git commit -m "descrizione commit"
    1. DIGITARE: git push -u origin master

  • Come aggiornare i file da locale su web?

    1. Aprire il terminale WSL
    1. DIGITARE: git init
    1. DIGITARE: git add .
    1. DIGITARE: git commit -m "descrizione commit"
    1. DIGITARE: git push

  • Come scaricare i file dal web su una cartella locale?

    1. Aprire il terminale WSL
    1. DIGITARE: git init
    1. DIGITARE: git remote add origin (link del repository)
    1. DIGITARE: git pull origin master
