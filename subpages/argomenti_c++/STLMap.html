⚜ std::map

Gli oggetti Map sono contenitori associativi che memorizzano gli elementi in modo mappato. Ogni elemento ha un valore chiave e un valore mappato. Nessun valore mappato può avere gli stessi valori chiave.

Gli elementi in un oggetto Map sono ordinati in base ai valori “Keys”.

Creazione

Per creare una lista std::map è necessario includere la libreria:

#include <map>

per dichiarare un oggetto di tipo std::map:

std::map<key_type, value_type> map_name = {{key1, value1},{key2, value2}, ...};
  • std::map → dichiarazione di un contenitore STL di tipo map
  • <key_type> → tipo di dato relativo alle chiavi della map
  • <value_type> → tipo di dato relativo ai valori della map
  • map_name → identificativo dell'oggetto
  • key1, key2, ... → chiavi della map
  • value1, value2, ... → valori della map

Operazioni base

insert()aggiunge un elementro (coppia chiave-valore) alla map
erase()rimuove un elemento o un range di elementi dalla map
clear()rimuove tutti gli elementi dalla map
find()ricerca nella map tramite un valore chiave
size()ritorna il numero di elementi della map
empty()ritorna true se la mappa è vuota

Le operazioni di base delle std::map sono:

  1. Aggiunta elemento.
  1. Accesso alle chiavi e valori.
  1. Rimozione elemento.

Aggiunta elemento

  • tramite l'operatore [].
  • tramite la funzione insert() e make_pair().
    insert() → permette di aggiungere una coppia di elementi nella map. make_pair() → permette di creare un'associazione tra chiave e valore.
std::map<int, std::string> countries;
  
// usando l'operatore []
countries[1] = "France";
countries[2] = "England";

// tramite insert() e make_pair()
countries.insert(make_pair(3, "Italy"));
countries.insert(make_pair(4, "Sweden"));

// Possibile anche aggiungere elementi associati alla stessa chiave
countries[5] = "Holland";
countries[5] = "Germany";

Accesso elemento

Per accedere agli elementi di una std::map:

  • tramite l'operatore [].
std::map<int, std::string> countries;

countries[1] = "France";
countries[2] = "England";

std::cout << countries[2];    // output --> England
  • tramite iterator.
// dichiarazione di un oggetto di tipo map
std::map<int, std::string> countries;

countries[1] = "France";
countries[2] = "England";
countries[3] = "Italy";
countries[4] = "Sweden";

// dichiarazione di un iteratore map
std::map<int, std::string>::iterator iter;

// tramite un iteratore è possibile accedere a tutti gli elementi della map e stamparli
for (iter = countries.begin(); iter != countries.end(); ++iter) {
    std::cout << iter->first << " - " << iter->second << std::endl;
}
Nota: countries.begin() → fa riferimento al primo elemento della map. countries.end() → fa riferimento all'ultimo elemento della map. Possibile anche una sintassi alternativa come:
std::cout << (*iter).second;

Rimozione elemento

  • rimozione di un singolo elemento con erase().
  • rimozione di tutti gli elementi con clear(). 
// dichiarazione di un oggetto di tipo map
std::map<int, std::string> countries;

countries[1] = "France";
countries[2] = "England";
countries[3] = "Italy";

// rimozione elemento di chiave 3
contries.erase(3);   // rimuoviamo Italy dalla map

// rimozione di tutti gli elementi
countries.clear();
