⚜ std::deque

In C++, STL Deque è un contenitore sequenziale che fornisce la funzionalità di una struttura di dati a coda doppia.

In una coda regolare, gli elementi vengono aggiunti dalla parte posteriore e rimossi dalla parte anteriore. Tuttavia, in una Deque, possiamo inserire e rimuovere elementi sia dalla parte anteriore che posteriore.

Creazione

Per creare una lista std::deque è necessario includere la libreria:

#include <deque>

per dichiarare un oggetto di tipo std::deque:

std::deque<type> elem;
  • std::deque → dichiarazione di un contenitore STL di tipo deque
  • <type> → tipo di dato della deque
  • elem → identificativo dell’oggetto

Operazioni base

MetodiDescrizione
push_back()inserisce elemento in coda
push_front()inserisce elemento in testa
pop_back()rimuove elemento dalla coda
pop_front()rimuove elemento dalla testa
front()ritorna il primo elemento
back()ritorna l’ultimo elemento
at()accede all’elemento nella posizione passata
size()ritorna la dimensione della deque
empty()ritorna true se la deque è vuota, false altrimenti

Le operazioni di base delle std::deque sono:

  1. Aggiunta elemento.
  1. Accesso elemento.
  1. Rimozione elemento.
Nota: come per la std::map è possibile applicare il concetto di iterator per eseguire le operazioni.

Aggiunta elemento

2 modi per aggiungere elementi alla std::deque

  • push_back() → inserisce un elemento in coda.
  • push_front() → inserisce un elemento in testa.
std::deque<int> values {5, 2};  

// aggiunge 3 in coda
values.push_back(3);

// aggiunge 7 in testa
values.push_front(7);

// la deque conterra': 7, 5, 2, 3

Accesso elemento

4 modi per accedere agli elementi di una std::deque:

  • front() per accedere al primo elemento. 
  • back() per accedere all’ultimo elemento. 
  • at()  per accedere all’i-esimo elemento.
  • possibile anche con operatore []
std::deque<int> values {1, 2, 3};
  
std::cout << values.front() << std::endl;   // output: 1
std::cout << values.back() << std::endl;    // output: 3
std::cout << values.at(1) << std::endl;     // output: 2
std::cout << values[1] << std::endl;        // output: 2

Rimozione elemento

2 modi per rimuovere elementi da una std::deque:

  • pop_back() → rimuove l’elemento in coda.
  • pop_front() → rimuove l’elemento in testa.
std::deque<int> values {1, 2, 3};

// rimuove elemento dalla coda
nums.pop_back();

// la deque conterra': 1, 2
  
// rimuove elemento dalla testa
nums.pop_front();

// la deque conterra': 2
