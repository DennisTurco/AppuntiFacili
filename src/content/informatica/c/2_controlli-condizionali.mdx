## 1. Descrizione

Un'istruzione if-else controlla il ramo condizionale. Le istruzioni nell'oggetto vengono eseguite solo se l'oggetto `if-branchcondition` restituisce un valore diverso da zero (o **`true`**). Se il valore di `condition` Ã¨ diverso da zero, l'istruzione seguente viene eseguita e l'istruzione che segue viene ignorata **`else`**. In caso contrario, l'istruzione seguente viene ignorata e, se Ã¨ presente un'istruzione **`else`** che segue, **`else`** viene eseguita.

## 2. Sintassi

La sintassi dell' `if` statement Ã¨:

```c
if (condizione) {
  // corpo dell'if statement
}
```

L' `if` statement valuta la `condizione` dentro alle parentesi `( )`.

- Se la `condizione` risulta `true`, il codice dentro al corpo dell' `if` viene *eseguito*.
- Se la `condizione` risulta `false`, il codice dentro al corpo dell' `if` viene *evitato*.

**Nota:** Il codice dentro `{ }` Ã¨ il corpo dell' `if` statement.

## 3. Funzionamento

L'istruzione **if** permette di verificare determinate condizioni ed ha la seguente sintassi:

```c
//...

if(condizione){
    //....
}

//...
```

In questo caso **se** l'espressione risulta **vera**, fa eseguire l'istruzione immediatamente successiva, **altrimenti** (se la condizione Ã¨ **falsa**) si salta l'istruzione (od il blocco di istruzioni) e si procede nell'esecuzione delle istruzioni successive, che possono essere la prosecuzione del programma o un semplice **else**, ad indicare la possibile alternativa all'if:

```c
if(condizione){
    //....
} else {
    //...
}
```

o un **else if** che permette di verificare una o piÃ¹ condizioni:

