## 1. Descrizione

Un **array monodimensionale** (o vettore) √® una sequenza di variabili dello stesso tipo che vengono situate consecutivamente nella memoria ed alle quali √® possibile accedere usando uno stesso nome (identificatore) a cui viene aggiunto un indice. Pensiamo all'**array** come ad un insieme di cassetti numerati.

**Definizione**: **Sequenza** di $n$ elementi ($n ‚â• 0$) tutti di tipo $t$ individuati singolarmente da un numero intero int fra 0 e $n-1$ (detto indice).

## 2. Dichiarazione

$$

$$

- $A$ ‚Üí identificatore (nome);
- $t$ ‚Üí tipo;
- $n$ ‚Üí costante intera, dimensione;

## 3. Dimensione Array

1. Numero elementi predefinito ‚Üí **array statici** = dimensione array costante (nota a compile-time)
2. Numero elementi specificato a run-time ‚Üí **array semidinamici** = dimensione array qualsiasi (intera)
3. Numero elementi non specificato ‚Üí **array dinamici**

<Aside variant="info">
Per gli array statici e semidinamici, una volta allocata, la dimensione dell'array non √® pi√π modificabile.

## 4. Operazioni

### Accesso elemento

```c

vett[4];  // -> mi riferisco al valore 3
```

### Lettura elemento

```c
int vett[5];
scanf("%d", &vett[2]); // inserisco un valore nella posizione 3
```

### Scrittura elemento

```c

printf("%d", vett[2]); //Output: 1
```

### Cosa NON si pu√≤ fare

- NO confronto e assegnamento

    ```c
    int A[10];
    int B[10];

    A == B; // ----> ERRORE
    A = B;  // ----> ERRORE
    ```

- NO I/O su array

    ```c
    int A[10];

    printf("%d", A); // ----> ERRORE
    scanf("%d", &A);  // ----> ERRORE
    ```

### Soluzione a questi problemi

per eseguire correttamente operazioni di confronto, assegnamento ed I/O √® spesso necessario utilizzare dei **cicli**.

- Confronto e assegnamento:

    *esempio*: `A[i] == B[j]`

    *esempio*: `A[i] = 5`

- I/O su array:

    *esempio*:  `scanf("%d", &A[3])`

    *esempio*:  `printf("%d", A[i])`

### Ordinamento Array (Bubble Sort)

![array4](https://giancarmelopittala.com/images/articles/bubble-sort.gif)

```c
int dim = 5;
int vett[] = {2, 0, 1, 6, 3};

for (int i = 0; i < dim; i++) {
        for (int j = 0; j < dim-1; j++) {
            if (vett[j] > vett[j+1]) {
                int temp = vett[j];
                vett[j] = vett[j+1];
                vett[j+1] = temp;
            }
        }
 }
```

### ESERCIZIO 2

Leggere un array di 10 interi e stampare solo i numeri che appaiono nell'array una volta soltanto.

> Esempio Output:
inserisci 10 numeri interi:  1 3 5 3 2 0 5 2 3 4
1 0 4
<details>
<summary>üí° Mostra soluzione</summary>

```c
#include <stdio.h>

int main(){
    int a[10];

    //memorizzo l'array
    printf("inserisci 10 numeri interi: ");
    for(int i=0; i<10; i++) {
        scanf("&d", &a[i]);
    }

    //per ogni numero dell'array
    for(int i=0;i<10;i++) {
        // voglio contare quante volte appare a[i]
        int volte=0;

        //controllo quante volte compare a[i]
        for(int j=0; j<10; j++) {
            if(a[j] == a[i]) {
                volte++;
            }
        }
        if(volte == 1) {
            printf("%d ", a[i]);
        }
    }
    return 0;
}
```

</details>
